- 工程化是如何做的？
- 构建流程是怎么样的？对于构建工具了解么？
- 上线流程是怎么样的？
- 本地开发和部署有什么区别？
- git 使用规范是怎么样的？怎么做好分支管理？
- 怎么做好团队协作？
- 前后端如何分工的？如何联调的？跨域如何做？
